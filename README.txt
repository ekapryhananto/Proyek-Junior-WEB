Name Project : Shop Computer

Description : Project ini dibuat untuk mengetahui data barang apa saja yang terdapat di Shop Computer. Project ini dibuat dengan laravel 8 yang di dalamnya bisa melakukan 
penambahan data, mengedit data, menghapus data, dan melihat detail barang yang ada. Data yang di inputkan akan di simpan dalam database "shopcomp", di dalam databse tersebut 
ada beberapa kolom yaitu id, nama, harga, deskripsi, foto, created_at, dan updated_at.

How To Use : download melalui github, ekstrak, dan buka lewat VC code.
